# Original Dashboard Mockups

These are the original self-contained HTML mockups the dashboard was designed
from. They are kept as **reference only** — the live app is the Next.js code in
`app/` and `components/`, not these files.

| File | Theme | Status |
| --- | --- | --- |
| `arctic-frost-dashboard.html` | Arctic Frost (dark + light) | Ported in Phase 0 — this is the canonical design. |
| `sunset-boulevard-dashboard.html` | Sunset Boulevard | Deferred. Will be added as a second selectable palette in a later phase. |

To map a mockup region to the implemented components, see
`docs/architecture/project-structure.md`.
